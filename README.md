# Spaceships
### Componente
```
 - Arduino Uno R3 x 1
 - USB Cable x 1
 - Breadboard x 2
 - Matrice 8x8 (1088AS) x 1
 - Driver MAX7219 x 1
 - LCD 16x2 x 1
 - Joystick x 1
 - Condensator ceramic 104 uF x 1
 - Condensator 10 uF x 1
 - Rezistenta 10k ohmi x 1
 - Rezistenta 220 ohmi x 1
 - Fire tata-mama
 - Fire tata-tata
```
### Descriere
```
  Spaceships este un joc ce tine de concentrare si agilitate.
  Poti alege sa eviti obstacolele sau sa le dobori.
  Incearca sa ramai in viata si sa faci un scor cat mai mare.
  Scorul se mareste de fiecare data cand impusti un obstacol.
  Cu cat obstacolul este mai mare cu atat aduni mai multe puncte.
  De fiecare data cand treci la un nivel nou viteza se mareste.
```
### Cum se joaca
```
  La inceputul jocului se va afisa mesajul "START GAME!" pe display.
  Pentru a incepe jocul trebuie apasat butonul de la joystick.
  In timpul jocului vor fi afisate pe display numarul de vieti si scorul.
  Ca sa te misti in stanga/dreapta trebuie miscat joystick-ul in stanga/dreapta.
  Pentru a distruge obstacolele trebuie apasat pe butonul joystick-ului.
  De fiecare data cand te loveste un obstacol vei pierde o viata.
  Daca ramai fara vieti iti pierzi scorul si va fi afisat pe display mesajul "GAME OVER".
  Daca reusesti sa treci de toate cele 3 nivele scorul ti se va actualiza si va fi afisat pe diplay.
  La finalul jocului vei fi intrebat daca vrei sa te joci jocul din nou.
  In caz afirmativ trebuie miscat de joystick in stanga si astfel jocul incepe din nou de la primul nivel.
  In caz negativ trebuie miscat de joystick in dreapta si astfel jocul se inchide.
```
